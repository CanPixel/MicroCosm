# MicroCosm

Welcome to MicroCosm, an interactive cellular ecosystem simulation built with Next.js in Firebase Studio.

## About The Game

In MicroCosm, you control a single biological cell adrift in a primordial soup. Your goal is to survive and thrive. Navigate the environment, consume nutrients to grow larger, and watch as the world evolves around you.

## How to Play

-   **Movement:** Use the **W, A, S, D** or **Arrow Keys** to move your cell.
-   **Growth:** Consume the small, white sugar crystals to increase your cell's size and score.
-   **Energy:** Movement drains your energy. Consuming sugar replenishes it. If your energy drops to zero, the game is over.

## Tech Stack

This project demonstrates building an interactive game using modern web technologies:

-   **Framework:** [Next.js](https://nextjs.org/) (with App Router)
-   **Language:** [TypeScript](https://www.typescriptlang.org/)
-   **UI Components:** [ShadCN UI](https://ui.shadcn.com/)
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
-   **Generative AI:** [Genkit](https://firebase.google.com/docs/genkit)
-   **Development Environment:** Prototyped in [Firebase Studio](https://firebase.google.com/docs/studio).

## Running the Project

To run the development server:

```bash
npm run dev
```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result. You can start editing the page by modifying `src/app/page.tsx`.
